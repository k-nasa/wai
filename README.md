## Usage

```bash
:) % wai -h
wai 0.1.0
k-nasa <htilcs1115@gmail.com>
A simple wasm interpreter

USAGE:
    wai [OPTIONS] <file-path> --invoke <invoke>

ARGS:
    <file-path>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -a, --args <args>...
    -i, --invoke <invoke>
```
